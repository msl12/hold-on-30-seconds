# Can you hold on more than 30 seconds?

a simple funny game.

## Install

```
git clone https://github.com/msl12/hold-on-30-seconds.git
cd hold-on-30-seconds/
npm install
npm start
```

Then open http://localhost:3000/ and have fun!
